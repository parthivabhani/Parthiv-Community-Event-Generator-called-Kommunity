Community Event Planner (Frontend)

This is the frontend of the Community Event Planner web app. It is built using React and Vite, and styled with Tailwind CSS. The interface allows users to discover, create, and manage community events with ease.

Tech Stack:
- React (Vite) for the UI
- Tailwind CSS for styling
- Axios for API requests
- React Router DOM for navigation

Setup and Run:
Navigate to the frontend folder:
cd frontend

Install dependencies:
npm install

Start the development server:
npm run dev

Build for production:
npm run build
